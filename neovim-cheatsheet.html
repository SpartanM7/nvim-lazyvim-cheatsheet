Foundations
🎭 Modes
iEnter Insert mode (before cursor)
aEnter Insert mode (after cursor)
AEnter Insert mode at end of line
oOpen new line below and enter Insert
OOpen new line above and enter Insert
vEnter Visual mode (character select)
VEnter Visual Line mode
Ctrl-vEnter Visual Block mode
:Enter Command mode
EscReturn to Normal mode (home base)
💾 Save & Quit
:wSave (write) file
:qQuit
:wqSave and quit
ZZSave and quit (shortcut)
:q!Quit without saving (force)
:qaQuit all open windows
Movement
🧭 Basic Movement
hMove left
jMove down
kMove up
lMove right
Ctrl-dScroll down half page
Ctrl-uScroll up half page
zzCenter screen on cursor
⚡ Word Motions
wJump to start of next word (stops at punctuation)
WJump to next WORD (whitespace-separated only)
bJump backward to start of word
BJump backward to start of WORD
eJump to end of word
EJump to end of WORD
📏 Line Motions
0Jump to absolute start of line
^Jump to first non-blank character
$Jump to end of line
f{char}Find — jump to next {char} on line
t{char}Till — jump just before next {char}
;Repeat last f/t jump (forward)
,Repeat last f/t jump (backward)
📄 File Motions
ggJump to top of file
GJump to bottom of file
{n}GJump to line n (e.g. 50G)
:{n}Jump to line n via command (e.g. :50)
Ctrl-oJump back in location history
Ctrl-iJump forward in location history
Editing — The Vim Grammar: [count] operator motion
✂️ Operators
dDelete (cuts to register)
cChange (delete + enter Insert mode)
yYank (copy)
pPaste after cursor
PPaste before cursor
uUndo
Ctrl-rRedo
🔥 Operator + Motion Combos
dwDelete to next word
d$Delete to end of line
ddDelete entire line
DDelete to end of line (shortcut for d$)
cwChange word
CChange to end of line (shortcut for c$)
ccChange entire line
yyYank (copy) entire line
2ddDelete 2 lines (prefix any op with count)
3wJump 3 words forward
🎯 Text Objects
iwInner word (just the word)
awAround word (word + trailing space)
i"Inner double quotes
a"Around double quotes (includes quotes)
i'Inner single quotes
i(Inner parentheses
a(Around parentheses (includes parens)
i{Inner curly braces
itInner HTML/XML tag
💡 Text Object Combos (most used)
ci"Change inside quotes → keeps quotes, replaces content
ca"Change around quotes → removes quotes too
di(Delete inside parentheses
dawDelete word including surrounding space
yiwYank (copy) just the word under cursor
ci{Change inside curly braces (e.g. function body)
vitVisually select inside HTML tag
Search & Replace
🔍 In-File Search
/patternSearch forward in file
?patternSearch backward in file
nNext match (same direction)
NPrevious match (reverse direction)
*Search for word under cursor (forward)
#Search for word under cursor (backward)
:nohClear search highlights
🔄 Search & Replace
:%s/old/new/gReplace all in file
:%s/old/new/gcReplace all with confirmation
:5,10s/old/new/gReplace in lines 5–10 only
'<,'>s/old/new/gReplace in visual selection
LazyVim — Leader Key Shortcuts (Space = Leader)
🔭 Telescope (Fuzzy Finder)
Space SpaceFind files by name in project
Space /Live grep across entire project
Space s gSearch → grep (same as Space /)
Space s wSearch for word under cursor across project
Space s sSearch document symbols (functions, classes)
Space s SSearch workspace symbols (entire project)
Space f rRecently opened files
Space f bFind open buffers
⌨️ Inside Telescope
TypeFuzzy filter results instantly
Ctrl-j / Ctrl-kMove down / up in results
EnterOpen selected file
Ctrl-vOpen in vertical split
Ctrl-xOpen in horizontal split
EscClose Telescope
🌳 Neo-tree (File Explorer)
Space eToggle file tree sidebar
Space EOpen tree focused on current file
j / kMove up/down in tree
EnterOpen file or expand folder
aAdd new file
dDelete file
rRename file
yCopy file
xCut file
pPaste file
/Filter/search within tree
qClose tree
📑 Buffers & Windows
Shift-lNext buffer (tab right)
Shift-hPrevious buffer (tab left)
Space b dClose / delete current buffer
Space b bVisual buffer picker
Ctrl-hMove focus to left window/split
Ctrl-lMove focus to right window/split
Ctrl-jMove focus to window below
Ctrl-kMove focus to window above
🧠 LSP — IDE Features
g dGo to Definition
g rGo to References (shows in Telescope)
g IGo to Implementation
KHover docs — type info + signature
Space c aCode Actions (fixes, imports, refactors)
Space c rRename symbol across entire project
[ dJump to previous diagnostic (error/warning)
] dJump to next diagnostic
Space x xShow diagnostics list for current file
💡 The Vim Grammar: [count] + operator + motion/text-object — learn 5 operators and 10 motions and you get 50+ commands for free. Example: 3dw = delete 3 words · ci" = change inside quotes · yiw = copy word under cursor.
